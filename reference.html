  • cumulative (bool) – if True and self.characters_graph is dynamic, plot a cumulative graph instead of a sequential one

  • graph_start_idx (int) – When self.characters_graph is dynamic, index of the first graph to plot, starting at 1 (not 0, since the graph slider starts at 1)

  • stable_layout (bool) – if self.characters_graph is dynamic and this parameter is True, characters will keep the same position in space at each timestep. Characters’ positions are based on the final cumulative graph layout.

    • if an int is given, the distance is in number of tokens
